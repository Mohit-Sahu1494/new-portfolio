# 🚀 Personal Portfolio

A modern and responsive personal portfolio website built using React.js and Tailwind CSS.

## ✨ Features

- Responsive Design
- Modern Glassmorphism UI
- Smooth Animations
- Projects Showcase
- Skills Section
- Contact Section
- Download Resume
- Social Media Links

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- Framer Motion
- Lucide React

## 📂 Project Structure

```
src/
├── components/
├── assets/
├── pages/
├── App.jsx
└── main.jsx
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Navigate to project directory

```bash
cd portfolio
```

3. Install dependencies

```bash
npm install
```

4. Run development server

```bash
npm run dev
```
